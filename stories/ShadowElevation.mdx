## Usage Guide

<h3>사용 예시</h3>
<p>테마에서 shadow 토큰을 가져와 컴포넌트에 적용합니다.</p>

<pre><code>{`import { shadow } from '@/tokens/theme';

// 카드 컴포넌트에 Level 1 그림자 적용
const Card = styled.View\`
  background-color: #fff;
  border-radius: 16px;
  \${shadow.level1}
\`;

// 스타일 객체로 직접 사용
const modalStyle = {
  backgroundColor: '#FFFFFF',
  borderRadius: 16,
  ...shadow.level3,
};

// React Native에서 사용
<View style={[styles.container, shadow.level2]}>
  <Text>플로팅 카드</Text>
</View>`}</code></pre>
