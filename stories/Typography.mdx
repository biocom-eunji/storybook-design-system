  <strong>React Native lineHeight 주의</strong><br />
  CSS의 <code>lineHeight: 1.5</code>는 폰트 크기의 1.5배이지만, React Native에서는 <strong>절대 px 값</strong>입니다.

  아래 토큰은 모두 <strong>절대 px 단위</strong>로 정의되어 있으므로 앱에 바로 적용 가능합니다.
