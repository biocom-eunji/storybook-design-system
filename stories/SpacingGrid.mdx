## Usage Guide

<h3>사용 예시</h3>
<p>테마에서 spacing, screenMargin, gridUnit 토큰을 가져와 사용합니다.</p>

<pre><code>{`import { spacing, screenMargin, gridUnit } from '@/tokens/theme';

// 컴포넌트 내부 패딩
const cardStyle = {
  padding: spacing.medium,       // 12px
  gap: spacing.small,            // 8px
};

// 화면 좌우 마진
const screenStyle = {
  paddingHorizontal: screenMargin.default,  // 16px
};

// 그리드 단위 활용 (커스텀 간격)
const customGap = gridUnit * 5;  // 20px
const customPadding = gridUnit * 6;  // 24px`}</code></pre>
